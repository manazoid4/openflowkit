# Monetisation

The free product must remain genuinely useful. Paid plans monetize managed convenience, not basic access.

## Free

- local transcription
- BYOK LLM refinement
- personal dictionary
- local snippets
- community support

## Pro

- managed fast STT
- managed LLM cleanup
- cloud sync
- advanced writing styles
- app-specific profiles
- higher monthly usage

## Teams

- shared dictionary
- shared snippets
- usage dashboard
- policy controls
- admin billing
- priority support

## Enterprise

- private deployment
- SSO/SAML/SCIM
- audit logs
- data retention controls
- dedicated model routing
- compliance support

## Gating principle

Never block the product shell. Gate:

- managed provider usage
- cloud sync
- team administration
- retention and audit controls
- enterprise deployment services

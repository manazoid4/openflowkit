## App surfaces

- Any text field through native paste injection.
- IDEs: Cursor, Windsurf, VS Code, JetBrains.
- Chat and LLM tools: ChatGPT, Claude, Gemini, Perplexity, OpenCode, Codex, Continue.
- Work apps: Gmail, Google Docs, Notion, Slack, Discord, Linear, GitHub.
- Browsers through extension-based command mode.

## Developer integration targets

- file mention detection
- symbol casing preservation
- terminal command formatting
- spoken diff and patch generation
- IDE extension API for richer context
